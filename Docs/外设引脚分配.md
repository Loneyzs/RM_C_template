- # RoboMaster 开发板 C 型硬件引脚分配表 (STM32F407IGH6)

  ## 1. 基础系统配置
  | 项目         | 参数          | 说明                     |
  | :----------- | :------------ | :----------------------- |
  | **MCU 型号** | STM32F407IGH6 | 高性能高性能基础 F4 系列 |
  | **内核频率** | 168 MHz       | 目标工作频率             |
  | **HSE 频率** | 12 MHz        | 外部高速晶振频率         |
  | **调试接口** | SWD           | SWDIO: PA13;SWCLK: PA14  |

  ---

  ## 2. 通用交互外设

  ### 2.1 用户 LED (RGB)
  *共阳极设计，由 PH10/11/12 控制 。*

  | 资源             | 引脚 (IO) | 极性       |
  | :--------------- | :-------- | :--------- |
  | **红灯 (LED_R)** | **PH12**  | 高电平点亮 |
  | **绿灯 (LED_G)** | **PH11**  | 高电平点亮 |
  | **蓝灯 (LED_B)** | **PH10**  | 高电平点亮 |

  ### 2.2 用户按键
  | 资源                 | 引脚 (IO) | 极性                  |
  | :------------------- | :-------- | :-------------------- |
  | **自定义按键 (KEY)** | **PA0**   | 低电平有效 (按下为低) |

  ---

  ## 3. 通信接口

  ### 3.1 CAN 总线
  | 资源     | 引脚 (TX / RX) | 说明                    |
  | :------- | :------------- | :---------------------- |
  | **CAN1** | **PD1 / PD0**  | 2-pin 接口              |
  | **CAN2** | **PB6 / PB5**  | 4-pin 接口 (带 5V 供电) |

  ### 3.2 UART 串口
  *注意：外壳丝印与芯片外设编号存在映射反转 。*

  | 丝印标识  | 芯片外设  | 引脚 (TX / RX) | 说明                       |
  | :-------- | :-------- | :------------- | :------------------------- |
  | **UART1** | **UART6** | **PG14 / PG9** | 3-pin 接口，常连接裁判系统 |
  | **UART2** | **UART1** | **PA9 / PB7**  | 4-pin 接口                 |

  ---

  ## 4. 动力与控制 (PWM)
  *7 路 PWM 接口总输出电流最大 5A ，位于 24-pin 扩展接口上。*

  | 资源通道 | 定时器通道 | 引脚 (IO) |
  | :------- | :--------- | :-------- |
  | **PWM1** | TIM1_CH1   | **PE9**   |
  | **PWM2** | TIM1_CH2   | **PE11**  |
  | **PWM3** | TIM1_CH3   | **PE13**  |
  | **PWM4** | TIM1_CH4   | **PE14**  |
  | **PWM5** | TIM8_CH1   | **PC6**   |
  | **PWM6** | TIM8_CH2   | **PI6**   |
  | **PWM7** | TIM8_CH3   | **PI7**   |

  ---

  ## 5. 板载传感器 (第三方驱动开发)

  ### 5.1 六轴 IMU (BMI088)
  *使用 SPI1 通信，支持最大 10MHz 速率 。*

  | 信号名         | 引脚 (IO) | 说明                  |
  | :------------- | :-------- | :-------------------- |
  | **SPI1_SCK**   | **PB3**   | SPI1 时钟线           |
  | **SPI1_MOSI**  | **PA7**   | SPI1 数据输出         |
  | **SPI1_MISO**  | **PB4**   | SPI1 数据输入         |
  | **CS1_Accel**  | **PA4**   | 加速度计片选 (低有效) |
  | **CS1_Gyro**   | **PB0**   | 陀螺仪片选 (低有效)   |
  | **INT1_Accel** | **PC4**   | 加速度计中断引脚      |
  | **INT1_Gyro**  | **PC5**   | 陀螺仪中断引脚        |

  ### 5.2 磁力计 (IST8310)
  *使用 I2C3 通信，默认 IIC 地址为 0x0E 。*

  | 信号名           | 引脚 (IO) | 说明             |
  | :--------------- | :-------- | :--------------- |
  | **I2C3_SCL**     | **PA8**   | I2C3 时钟线      |
  | **I2C3_SDA**     | **PC9**   | I2C3 数据线      |
  | **RSTN_IST8310** | **PG6**   | 硬件复位引脚     |
  | **DRDY_IST8310** | **PG3**   | 数据就绪中断引脚 |

  ### 5.3 IMU 恒温加热电路
  *通过 PWM 控制功率，提升陀螺仪在不同温度下的稳定性 。*

  | 资源         | 定时器通道    | [cite_start]引脚 (IO) | 加热功率             |
  | :----------- | :------------ | :-------------------- | :------------------- |
  | **加热控制** | **TIM10_CH1** | **PF6**               | 高电平满功率 (0.58W) |
